# Store Collection - Tienda Online

Tienda online minimalista y elegante hecha con **Angular 18 (standalone)** + **Spring Boot 3**.

### Accesos rápidos

**Parte pública (catálogo)** → sin imágenes aún  
http://localhost:4200

**Panel administrador**  
http://localhost:4200/admin  
(credenciales: pedir al dueño)

**Swagger API (Spring Boot)**  
http://localhost:8080/swagger-ui/index.html

### Cómo correr

```bash
# Backend
cd backend
./mvnw spring-boot:run   (o mvn spring-boot:run)

# Frontend
cd frontend
npm install
npm start
